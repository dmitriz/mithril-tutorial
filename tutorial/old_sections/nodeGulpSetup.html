Ok, we're ready to start building our app.

Now that some of the basics are out of the way, let's get ourselves into a module-based environment.

Feel free to put together your build system of choice, or follow these instructions to use my gulp/browserify boilerplate:

  1. If you haven't already, download and install Node.js (includes npm).
  2. Once Node.js is installed, download and unzip the blank mithril_tutorial_1 folder. It currently contains only the .gulpfile and package.json file you'll need, along with a near-empty src folder. You can move the mithril_tutorial_1 folder to your preferred working projects location.
  3. In a terminal window, cd into mithril_tutorial_1, then type 
    npm install
  4. After the installation has finished, type 
    gulp
  5. Open a new tab in terminal, cd into mithril_tutorial_1/build, then type 
    live-server
    You should now have a new, empty window or tab in your default browser.
  6. The last step is to bring your gulp tab in Terminal to the foreground. Bad errors in CSS or JavaScript can crash the build process, so if that happens, you'll need to gulp again after you've fixed your code.
